# 1. 컨테이너 안에는 별도의 리눅스 환경이 설치됩니다. 
# /디렉토리 밑에 app 이라는 디렉토리를 만들어서 그 안에 모든 파일을 옮겨둡니다.

# 2. 컨테이너 안은 완전 격리된 환경입니다. (host.docker.internal) 

# 3. 이미지 안에는 불필요한 파일, 중요한 파일들은 넣지 않습니다. (.dockerignore 파일에 작성합니다)

# 4. 도커이미지에 올림 (docker tag 기존이미지명 새로운이미지명 -> docker push 새로운이미지명)
my-fastapi-app:1.0 -> [레지스트리-id명0]/my-fastapi-app:1.0

# 5. db나 로그파일 같은 것들은 이미지 밖에 별도의 저장소를 사용합니다.

# 6. 이미지는 가벼울수록 빌드,배포,업그레이드에 시간이 단축됩니다.
